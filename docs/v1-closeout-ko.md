# Wine Cellar Scan v1 종료 메모

작성일: 2026-04-05

## 종료 정의

이 문서는 `Wine Cellar Scan`의 `v1 종료 시점`을 기록합니다.

여기서 종료는 프로젝트 폐기가 아니라, 현재 MVP 범위를 닫고 다음 확장은 별도 장기 계획으로 넘긴다는 의미입니다.

## v1에서 마감한 범위

- 병 라벨 사진 업로드와 OCR 추출
- 후보 와인 추천 및 수동 선택
- 개인 시음 기록 저장
- `또 먹을 와인 / 다시 안 먹을 와인` 분류
- 셀러 카드, 정렬, 페이지네이션, 기본 인사이트
- 기본 샘플 데이터 시드
- 일부 와인에 대한 실제 라벨 이미지 고정 표시
- 파비콘 / 앱 아이콘 반영
- 공개 저장소용 API 키 가이드 정리

## v1에서 의도적으로 닫은 범위

- 사용자 계정 / 인증
- 멀티 유저 데이터 분리
- 와인 병 수량 관리
- 실제 상용 와인 DB 정식 연동
- 비용 제어가 포함된 OCR 운영 체계
- 모바일 앱 패키징
- 결제 / 구독

## 운영 원칙

- 공개 저장소에는 실제 운영용 API 키를 남기지 않는다.
- 클론 사용자는 각자 `.env`에 자신의 키를 넣어야 live OCR을 사용할 수 있다.
- 런타임 데이터는 시드 데이터와 분리한다.

## 종료 후 유지 항목

- 버그 수정이나 문서 보완은 가능하다.
- 장기 제품화 아이디어는 `docs/long-term-plan-ko.md`에서 별도로 관리한다.
- 이 문서는 `v1 종료 기준점`으로 유지한다.
